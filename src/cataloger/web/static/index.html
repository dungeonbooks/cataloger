ISBNs (one per line)

Paste ISBN-10 or ISBN-13 codes, one per line. Hyphens are stripped automatically. Max 100 per batch.

How it works

  1. Enter your Store Location name — this must match your Square Location exactly so the CSV maps items to the right store. You can find it at Square Dashboard → Locations.
  2. Paste your ISBNs — one per line. You can get these from barcode scanners, spreadsheets, or type them in manually.
  3. Generate — we look up each ISBN to find the title, author, description, cover image, and price.
  4. Download CSV — import this file into Square: go to Square Dashboard → Items → Import Items and upload the CSV.
  5. Download Images — a ZIP of cover images named by ISBN. You'll need to upload these to each item manually in Square since image upload requires a connected Square account (coming soon).

About cover images

Cover images are included in the download but cannot be imported via CSV — Square doesn't support that. To add images, you'd need to upload them one-by-one in the Square Dashboard, or connect your Square account so we can upload them via the API. Square account connection is planned for a future update.
